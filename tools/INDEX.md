# Agent tools
- solana_wallet_intel: Analyzes Solana wallet SOL/SPL balances, USD valuation, and activity from provided on-chain data; classifies tier and activity level.
- soltoethconverter: Converts an amount of SOL to ETH using USD prices (solPriceUsd/ethPriceUsd) or a direct ethPerSol rate. Returns rate, ETH amount and USD value.
